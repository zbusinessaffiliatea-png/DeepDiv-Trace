Security Policy

Which Versions We Cover

We only push security updates to the current main branch. If you're running an old fork or an outdated version, we can't guarantee you're protected. Keep it fresh.

Found Something? Tell Us.

If you stumble across a vulnerability in DeepDiv-Trace—whether you're a security researcher, a curious dev, or just poking around—please don't drop it in public. Shoot us an email at [aims.ssa.369@gmail.com] or DM us on X @DeepDiv.

Here's What You Can Expect From Us:

· 24 hours: You'll hear back from us. No ghosting.
· A timeline: We'll let you know when we plan to ship a fix.
· Credit: If your find is legit, you're going on the contributors list. You earned it.

A quick note: We're community-powered right now. We're working on setting up a proper bug bounty fund, but for now, what we can offer is respect, recognition, and our genuine gratitude.
